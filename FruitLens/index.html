Description of Project

This project takes in an input of an image and uses machine learning technology to indentify what fruit is being displayed. Juan took the approach of creating the front end development by creating the website and its functionality. Ryan decided on coding and training a machine learning model that would run in the back end to identify whatever picture it got sent.
